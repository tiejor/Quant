# ClaudeQuant

A股因子模型验证系统 — 自定义因子、截面回测、因子评估。

## Language

### 核心概念

**因子（Factor）**：衡量股票在某截面上相对强弱的标量。输入当日横截面数据，输出每只股票一个数值。用户自定义因子 = 在 `factors.py` 中写一个 Python 函数。
_Avoid_: 信号、特征、指标

**截面回测（Cross-Sectional Backtest）**：在每期期末按因子值对所有股票排序分组，在下一持仓期内等权持有各组，考察各组未来收益差异。核心问题："因子值的高低能否区分未来收益的高低？"
_Avoid_: 时间序列回测、择时回测

**股票池（Universe）**：回测当日在调仓日有资格进入分组的股票集合。只在调仓日更新。过滤规则：排除 ST 状态、排除上市不足 365 天的新股。停牌不做剔除。
_Avoid_: 全市场、候选池

### 时间节点

**因子计算日（Factor Calc Date）**：每月最后一个交易日。使用当月及历史数据计算因子值、确定分组边界。多轨道偏移不改变因子计算日。

**调仓日（Rebalance Date）**：新持仓实际生效的交易日。无偏移时为下月第一个交易日；轨道偏移 k 天后顺延 k 个交易日执行。
_Avoid_: 建仓日（建仓只覆盖"开新"，调仓包含"平旧 + 开新"）

**持仓期（Holding Period）**：从本次调仓日到下次调仓日前一个交易日之间的时间段。持仓期内分组不变，每日对各组内股票做等权收益率平均。月频下，因子计算日固定为每月末最后一个交易日，调仓日决定了持仓期的起止边界。
_Avoid_: 窗口、区间

### 因子预处理

**MAD 标准化（MAD Standardization）**：每个截面内，用中位数绝对离差（Median Absolute Deviation）将因子值标准化。`(因子值 - 截面中位数) / MAD`，其中 `MAD = median(|因子值 - 截面中位数|)`。比 z-score 更抗极端值。默认开启，可通过参数跳过。
_Avoid_: 归一化、z-score

### 回测概念

**分组（Group）**：在因子计算日按因子值从高到低将股票池分成 N 等份（默认 10 组）。第 1 组为因子值最高的股票，第 N 组为最低。分组边界由当日因子值分位数决定，每组股票数量相等（或差 1）。
_Avoid_: 分位、档次

**分组收益（Group Return）**：持仓期内，每天对各组内所有股票的个股日收益率做等权平均，得到该组当日收益率 $r_{组,t}$。
_Avoid_: 组合收益（分组收益特指因子分组后的等权组合）

**多空组合（Long-Short Portfolio）**：每日做多最高分组、做空最低分组：$r_{多空,t} = r_{G_1,t} - r_{G_N,t}$。代表因子区分收益能力的纯粹暴露，不包含市场 beta 对冲。
_Avoid_: 对冲组合

### 因子评估指标

**Rank IC**：单个截面内，因子值与未来收益的 Spearman 秩相关系数。均值衡量因子预测方向与力度，正值表示因子值高的股票未来收益更高。
_Avoid_: Pearson IC

**IR（Information Ratio）**：Rank IC 序列的均值除以标准差（IC_mean / IC_std），衡量因子预测稳定性。IR 越高，IC 在各期之间波动越小。
_Avoid_: 信息系数（IC 和 IR 是不同的量）

**IC 半衰期（IC Half-Life）**：IC 衰减到初始值一半所需的期数。从因子计算日向后逐期计算滞后 IC，半衰期越长，因子有效期越久。

### 稳健性检验

**轨道（Pathway）**：对调仓日序列偏移 k 个交易日后生成的一条独立回测路径。因子计算日不变（始终为月末），偏移只影响调仓执行日。

**轨道偏移（Pathway Delay）**：生成 n+1 条轨道（偏移 0, 1, ..., n 个交易日），偏移 0 为基准轨道，每条轨道独立回测。检验结果是否依赖特定建仓日：因子稳健则各轨道指标接近；方差大则调仓日选择对结果影响显著。
_Avoid_: 敏感性测试

## Relationships

- **因子计算日** 决定 **因子** 的取值截面
- **因子** 在 **股票池** 上计算，按因子值排序产生 **分组**
- **分组** 在 **调仓日** 生效，开始 **持仓期**
- **持仓期** 内逐日追踪得到 **分组收益**
- 最高组与最低组的分组收益差 = **多空组合** 收益
- 每期 **Rank IC** 构成 IC 序列，IC 序列衍生出 **IR**、**IC 半衰期**
- **轨道偏移** 对调仓日序列做 shift(k)，生成多条 **轨道**，每条轨道独立评估以上所有指标

## Flagged ambiguities

- 股票池只在调仓日更新，月内若有股票退市，仍停留在分组中直到下次调仓。v1 不做月内逐日退出处理。
- 停牌股票不做特殊处理——若调仓日股票停牌，仍进入分组；若整个持仓期停牌，收益率为 NaN 或 0，对等权组合影响有限。
