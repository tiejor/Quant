# Triage Labels

本 repo 使用 Matt Pocock 五标签 triage 体系。

| 角色 | 标签 | 含义 |
|------|------|------|
| 待评估 | `needs-triage` | 维护者需评估此 Issue |
| 等待回复 | `needs-info` | 等待报告者提供更多信息 |
| AFK 就绪 | `ready-for-agent` | 完全明确，可由 AFK agent 独立执行 |
| 需人工 | `ready-for-human` | 需要人类介入实现 |
| 不修复 | `wontfix` | 不做处理 |

当技能提及 triage 角色时（如"打上 AFK-ready 标签"），使用上述对应的标签字符串。
